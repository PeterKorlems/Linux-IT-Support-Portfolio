## Assignment 3: Log and Configuration File Search

### Scenario
An application is failing to start correctly.
You are asked to locate relevant configuration files and search for potential error indicators.

### Objectives
- Locate files efficiently
- Search file contents for keywords
- Demonstrate troubleshooting methodology

### Tasks
1. Search the system for configuration files ending in `.conf`
2. Identify at least one configuration file in `/etc`
3. Search for the following keywords (case-insensitive where appropriate):
   - `error`
   - `name`
4. Record your findings in a file named:
   - `search_results.txt`

### Commands You Are Expected to Use
- `find`
- `grep`
- `cat` or `less`

### Evidence to Include
- Commands used for searching
- Example results found
- Contents of `search_results.txt`

### IT Support Skills Demonstrated
- Efficient troubleshooting techniques
- Log and configuration file analysis
- Problem isolation and investigation skills
