# Linux Basics – Practical Challenge I

## Overview
This assignment is designed to consolidate Linux basics knowledge and demonstrate
practical IT support skills aligned with an IT Support Associate role.

---

## Assignment 1: User Workspace Investigation

### Scenario
A user reports that files they recently worked on appear to be missing from their home directory.
You have been asked to investigate the user’s workspace and confirm the directory structure
and file locations.

### Objectives
- Navigate the filesystem confidently
- Verify file and directory presence
- Demonstrate safe inspection practices

### Task breakdown
1. Confirm your current working directory
2. List all visible and hidden files in the home directory
3. Create a structured workspace for investigation:
   - `~/linux_basics_lab/investigation`
4. Create two files inside the investigation directory:
   - `findings.txt`
   - `notes.txt`
5. Add a short note to `findings.txt` describing what you checked

### Commands Used
- `pwd`
- `ls`, `ls -la`
- `cd`
- `mkdir`
- `touch`
- `echo`
- `cat` or `less`

### Evidence to Include
- List of commands used
- Confirmation of directory structure
- Contents of `findings.txt`

### IT Support Skills Demonstrated
- File system navigation
- Safe inspection of user directories
- Clear documentation of investigative steps
